Documentação CEP Bemean

Conforme esse projeto for evoluindo, documentaremos com alguns exemplos de utilização da api para que assim vocês tirem um bom proveito da api. Para usar é bem simples, basta seguir os exemplos abaixo, em caso de duvida ou sugestões, entrar em contato.

Webservice de CEP

Envie um CEP nos seguintes formatos com {8} ou {9} digitos. Utilize os seguintes exemplos.

https://cep-bemean.herokuapp.com/api/br/36500000
https://cep-bemean.herokuapp.com/api/br/36500-000
Retorno em Json: 

                        
{
    "code": "88370-410",
    "state": "SC",
    "district": "Centro",
    "address": "Rua Ezequiel Antero Rocha",
    "city": "Navegantes"
}

CEP não encontrado ou retornamos um desatualizado?
